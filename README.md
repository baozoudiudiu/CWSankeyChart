# CWSankeyChart
简易版桑基图表 Sankey Chart

主流的一些Chart库过于炫酷，而且一导入就把所有图表功能都导入到项目了，过于笨重。
于是自己撸了一个满足需求的简易版。
基于 UIView + CAShapeLayer + UIBezierPath 实现
